# Erasmus 2026 - Antwerpen

This repository contains files for the Erasmus Course en metaheuristics in Antwerp.

## Directory structure

This repository contains two folders:
- JuliaIntro
- Course

`JuliaIntro` contains the slides I used for the presentation shown during the classes. You can report to the running-scenario.pdf to see what I did for the demo. In case you are lost, you can always follow the video for the demo:

https://youtu.be/3zaBv8OR6yc

`Course` contains the source for the lab sessions. Please read the usage.md file that explain what we did together. 

Remember that the teacher's file contains all the operational code, but without the comments (that are in the student's version).
